#  «Smart Table»

Проект, посвящённый работе с **DOM-структурой страницы**, извлечению данных и их отображению в интерактивной таблице.  

Основной фокус — производительная работа с DOM, грамотная декомпозиция логики и модульная архитектура.

##  Запуск проекта

### Локальный запуск

1. Склонируйте репозиторий:
   ```bash
   git clone <https://github.com/a2soloveyy/smart-table.git>
   ```
2. Откройте файл index.html в браузере

3. Откройте DevTools → Console, чтобы увидеть результат работы парсера

**Режим разработки (Vite)**
```bash
npm install
npm run dev
```

**Проект будет доступен по адресу, указанному в терминале**

##  Реализованный функционал

- Парсинг DOM-структуры страницы
- Формирование единого объекта данных
- Отображение данных в таблице
- Поиск по нескольким полям
- Фильтрация данных
- Сортировка по значениям
- Пагинация с навигацией
- Перерисовка без повторных обработчиков событий
